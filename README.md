## ğŸ§  Purpose of NexusGen
The goal of NexusGen is to create a system where:

1. **I upload documents** (PDFs, notes, text files)  
2. The system **chunks and embeds** them  
3. A **vector store** holds these embeddings  
4. I can ask questions  
5. The system retrieves relevant chunks and uses **RAG** to give accurate answers  

This helps me understand how real-world AI systems are built internally.
